# Run it
```
docker run \
	-it \
	--rm \
	-e APP_ZIP_CODE=91405 \
	-e APP_PRODUCTS="MP062LL/A,MNPJ2LL/A" \
	-e APP_NOTIFY_EMAIL="your@email.com" \
	icheko/apple-product-availability
```

# Kill it
`control + c`
